# Contributing to Hive-Reflex

Thank you for your interest in contributing to Hive-Reflex! We welcome contributions from everyone.

## Code of Conduct

This project adheres to a Code of Conduct. By participating, you are expected to uphold this code.

## How to Contribute

### Reporting Bugs

1. Check if the issue has already been reported.
2. If not, open a new issue using the "Bug Report" template.
3. Provide a clear description and steps to reproduce.

### Suggesting Enhancements

1. Open a new issue using the "Feature Request" template.
2. Explain why this enhancement would be useful.

### Pull Requests

1. Fork the repo and create your branch from `main`.
2. Ensure your code follows the existing style.
3. Add tests for any new functionality.
4. Ensure all tests pass.
5. Create a Pull Request.

## Development Setup

Please refer to `docs/SETUP_GUIDE.md` for setting up your development environment.
